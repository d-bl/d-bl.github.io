These old generators may still have a use case but they are no longer maintained.

interactive web page

Limited set of stitches. Fixed set of patterns. Thread diagrams only from a table of stitches. A dedicated InkScape plugin complements the web application. The plugin can change the thread styles of downloaded diagrams, it is not upgraded to Inkscape 1.0.

desktop application

Only about a dozen traditional grounds and braids. An export of stitch by stitch images can be imported by third party tools to create an animated image that show a working order. You can try to execute it with OpenJDK 8.

interactive web page

Flanders only. No Pair diagrams. Works even with old browsers.
