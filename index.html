DiBL : Diagrams for Bobbin Lace

A collection of free software and web based tools related to bobbin lace.

The InkScape plugins are bundled together in a single installation package. InkScape is a powerful general purpose vector based drawing tool, like these bobbin lace tools it is free of charge and free to change or improve at your own discretion.

Diagrams

The screenshot snippets show either pair diagrams or thread diagrams of a rose ground, which differs from a Flanders ground by using four pairs per pin versus two pairs per pin.

GroundForge is a toolbox to design bobbin lace grounds or braids.
