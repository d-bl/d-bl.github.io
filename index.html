DiBL : Diagrams for Bobbin Lace

Two groups of freely available generators: diagrams and grids.

Each application in a group has its own use case. The InkScape plugins in both groups are bundled together in a single installation package. InkScape is a powerful general purpose drawing tool, like these bobbin lace tools it is free of charge and free to change or improve at your own discretion.

Diagram generators

The screenshot snippets show diagrams of Flanders lace, which differs from a Torchon rose ground by using four pairs per pin versus two pairs per pin. As far as available a thread diagram with only half stitches is shown, otherwise a pair diagram.

GroundForge

The ultimate toolbox to design bobbin lace grounds, though its predecessors still have some purpose.

Matching color coded pair diagrams, thread diagrams and patterns in a single view, supports any stitch. Apply thread styles on the browser canvas. So far only patterns with perpendicular lines.

Pair and thread diagrams for half a dozen of traditional grounds and half a dozen of braids. An export of step by step images can be imported by third party tools to create an animated image for a pattern. Fixed patch sizes that are relatively small. Replacing stitches involves the following steps for each stitch: look up the desired stitch, copy it, look up the stitch to replace (with a visual check), paste.
