Propportional effects of provided applications.
web More patterns that scale better. You can design your own patterns, though the definitions might be a bit cryptic.
InkScape plugin Allows matching nested grids . Less patterns but you can add your own marks for one pie and have it automagically repeated to a complete circle.

Diagram generators

screenshot snippets of flanders thread diagrams with only half stitches

web Matching pair and thread diagrams in a single view, any stitch. So far it provides the 2012 patterns with perpendicular lines of TesseLace. Apply thread styles on the browser canvas. Not resizable to a pricking as the pins choose their own position what might be off a traditional grid, rather use the diagrams as cheat-sheets alias voodoo boards.
web + InkScape plugin Matching pair and thread diagrams accessible with a few clicks. TesseLace patterns with perpendicular lines don't have a thread diagram. Limited sets of stitches, some grounds more than others. Generated diagrams can be downloaded with a right click. Apply thread styles with an InkScape plugin or pencil and tracing paper.
desktop Pair and thread diagrams for half a dozen of traditional grounds and half a dozen of braids. An export of step by step images can be imported by third party tools to create an animated image for a pattern. Fixed patch sizes that are relatively small. Replacing stitches involves the following steps for each stitch: look up the desired stitch, copy it, look up the stitch to replace (with a visual check), paste.
web Flanders only. No Pair diagrams. Apply thread styles with tracing paper and pencil.
